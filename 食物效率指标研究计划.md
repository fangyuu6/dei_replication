# 美味效率指数：弥合可持续食物指标的最后一块空白

## 研究计划 — 目标期刊：*Nature*（Article）

> **一句话定位**：感官科学量化"好吃"已有 60 年，可持续性科学量化食物的环境代价已有 20 年。但从来没有人把两者做过除法。我们提出首个将食物好吃程度（享乐价值）与资源消耗联系起来的标准化框架——**美味效率指数（Deliciousness Efficiency Index, DEI）**——并在全球尺度上进行验证。

---

## 1. 空白：一个本应存在却尚未诞生的指标

### 1.1 核心问题

我们已经拥有成熟的、经过验证的系统来衡量一种食物造成了多大的环境危害（生命周期评估 LCA、水足迹、碳足迹，参照 ISO 14040/14044、ISO 14067）。我们同样拥有成熟的、经过验证的系统来衡量一种食物有多好吃（1957 年诞生的 9 分享乐量表、ISO TC34/SC12 框架下的 31 项标准）。**但从来没有人把它们正式结合过。**

现有的每一个复合指标都将"营养"与"环境影响"配对——营养密度-气候影响指数（NDCI）、膳食环境指数（DEX）、可持续营养丰富食品指数——却没有一个纳入真正驱动食物选择的维度：**好吃程度（享乐价值）**。

Clark 等人（2022，*PNAS*）在对 57,000 种食品的研究中明确指出了这一空白：*"环境、营养与食物消费相关的更广泛结果（如经济成本、**可接受性、愉悦感**）之间是否存在权衡，仍不清楚。"*

### 1.2 为什么这在全球尺度上至关重要

| 事实 | 来源 |
|------|------|
| 商业厨房单位面积能耗是其他商业建筑的 **5–7 倍** | 美国能源部 |
| 烤箱和灶具中只有约 **40% 的能量**到达食物，60% 以废热形式散失 | 行业数据 |
| 中国外卖市场规模达 **1.64 万亿人民币**（2,290 亿美元），5.5 亿用户，估计每年产生 **370 亿个**塑料容器 | 美团/艾瑞 2024 |
| 中国餐饮人均每餐食物浪费 **74.16 克** → 年均 2,373 万吨 → 4,733 万吨 CO₂ 当量 | 2023 全国调查，n=1,957 |
| 每投入 **1 美元**用于减少食物浪费，餐厅可节省 **7 美元**运营成本 | Champions 12.3/WRAP/WRI，n=100+ 餐厅，12 个国家 |
| **73–80% 的消费者**愿意为可持续餐饮支付溢价 | 多项调查 |
| UCLA 每道菜碳标签使低碳食物选择增加了 **54%** | UCLA 干预研究 |

这个空白不是学术琐事。它直接关系到 **3.5 万亿美元的全球餐饮产业**、气候政策（食物系统占全球温室气体排放的 26–34%），以及 80 亿人类——他们选择吃什么，首先基于的是好不好吃，而不是营养密度评分。

### 1.3 为什么发 *Nature*

本研究满足 Nature 的核心标准：

1. **突出的科学重要性**：首次将两个成熟但隔离的测量传统（感官科学 + 可持续性科学）正式整合
2. **跨学科意义**：桥接食品科学、环境科学、行为经济学和公共政策
3. **广泛关注度**：食物是最普遍的人类体验，可持续性是本世纪的核心挑战
4. **可操作性**：该指标对产业、消费者和监管者有直接的实际应用价值
5. **范式转换**：将整个"可持续食物"的讨论从"什么营养又环保"重新定义为"什么好吃又环保"——与人类真实的食物决策方式对齐

### 2.2 现有复合指标为何都遗漏了"好吃"

| 指标 | 公式 | 缺少什么 |
|------|------|---------|
| NDCI（营养密度-气候影响指数） | 营养密度 / CO₂ 当量 | **好吃程度** — 没人吃的营养食物就是被浪费的食物 |
| DEX（膳食环境指数） | 加权环境得分 / 膳食 | **好吃程度** — 忽略了膳食是否可口 |
| 可持续 NRF 指数 | 营养丰富食品得分 / 温室气体 | **好吃程度** — 同样的问题 |
| EAT-Lancet 行星健康膳食 | 行星边界内的营养充足性 | **好吃程度** — 该方案因"不好吃"而被广泛批评 |
| NutriScore（A–E 分级） | 营养质量 | 既没有好吃程度，也没有环境维度 |

**根本原因**：营养科学和环境科学都使用客观的、可在实验室测量的量。感官科学产生的是"主观"数据（享乐评分），这两个领域历来认为其"不够硬"。但事实上，**经过规范施测的享乐评分在统计上与营养测量同样可靠**——数千项研究已经证明了这一点——而且它们能预测真正重要的结果：**人们到底吃不吃这个食物。**

### 2.3 行为学论证：驱动选择的是好吃，不是营养

仅含营养维度的复合指标存在根本性缺陷：人们不是基于营养密度来选择食物的。多项荟萃分析一致表明，**好吃程度是食物选择的第一预测因子**，排在价格、健康和便利性之前（Glanz 等，1998，在 23 个国家复制验证）。任何忽略其试图改变的行为的首要驱动力的可持续性指标，在结构上就是不完整的。DEI 将享乐价值——真正的决策变量——放在分子中，修正了这一缺陷。

## 3. DEI 框架：正式定义

### 3.1 核心公式

```
           H(f, p, c)
DEI(f) = ─────────────
           E(f, p, c)
```

其中：
- **f** = 食物品项（具体菜品或产品）
- **p** = 制作方式（食谱、烹饪技法、呈现方式）
- **c** = 情境（餐厅类型、地理区域、文化背景）
- **H** = 享乐价值函数（每份食物的好吃满意度）
- **E** = 环境代价函数（每份食物的资源消耗）

### 3.2 分子：享乐价值函数 H(f, p, c)

```
H(f, p, c) = μ_hedonic(f, p, c)
```

**核心测量**：标准化享乐量表上人群评分的均值。

| 量表选项 | 优势 | 局限 | 建议 |
|---------|------|------|------|
| 9 分享乐量表 | 通用性强，60+ 年验证 | 天花板效应，等级而非比率数据 | **消费者研究的主量表** |
| 标签享乐量表（LHS） | 比率水平数据，避免天花板效应 | 知名度较低 | 交叉验证量表 |
| 消费者平台评分（大众点评、Yelp、Google） | 样本量巨大，生态效度高 | 混杂了食物、服务和环境 | **大规模代理指标**，需去偏处理 |

**关键方法论决策**：

1. **人群水平而非个体水平**：DEI 使用人群享乐评分均值，而非个人评分。这消除了"口味是主观的"这一反对意见——当 n>30 时，享乐均值高度稳定且可重复。
2. **情境分层**：评分按文化情境分层（中国、西方、日本等），以考虑已记录的跨文化评分差异。
3. **平台数据作为代理**：在大规模分析（研究二）中，使用经过机器学习去偏的餐厅评论分数作为享乐代理指标，并通过对照感官品评数据进行验证。

### 3.3 分母：环境代价函数 E(f, p, c)

```
E(f, p, c) = Σᵢ wᵢ · Impactᵢ(f, p, c)
```

跨多个影响类别聚合，使用既有的 LCIA 加权方法：

| 影响类别 (i) | 单位 | 数据来源 | 范围 |
|------------|------|---------|------|
| 碳足迹 | kg CO₂ 当量/份 | Klimato、Poore & Nemecek 2018 | 从摇篮到餐桌 |
| 水足迹 | 升/份 | 水足迹网络 | 绿水 + 蓝水 + 灰水 |
| 能源消耗 | kWh/份 | MECS 项目、餐厅分项计量 | 烹饪 + 服务 + 清洗 |
| 固体废弃物 | kg/份 | Winnow 智能秤、废弃物审计 | 食物废弃 + 包装 |
| 洗涤负荷 | 升水 + kWh 能源/份 | 商用洗碗机规格、餐具数量 | 每道菜的清洗开销 |

**加权方法**：多准则决策分析（MCDA），使用层次分析法（AHP）和专家组评判，交叉验证 ReCiPe 2016 终点加权和产品环境足迹（PEF）标准化方法。

**标准化**：每个影响类别相对于该食物类别内的最大观测值归一化到 0–1，然后加权求和，得到无量纲的 E 分数。

### 3.4 DEI 分级

| DEI 等级 | 含义 | 类比 |
|---------|------|------|
| **DEI-A**（前 20%） | 卓越：很好吃且资源代价低 | 能效标识一级 / NutriScore A |
| **DEI-B**（20–40%） | 良好：高于平均效率 | |
| **DEI-C**（40–60%） | 中等：平均水平 | |
| **DEI-D**（60–80%） | 偏低：资源消耗相对于好吃程度偏高 | |
| **DEI-E**（后 20%） | 低效：高资源代价，享乐回报不成比例地低 | |

> 等级阈值由全球食物数据集的分布经验确定，而非人为设置。

### 3.5 呈现悖论——在框架内自然解决

一个关键挑战：**食物呈现方式确实会改善口感体验**（Michel 等，2014，*Flavour*：艺术摆盘 → 好吃评分 +18%，支付意愿翻倍；Zellner 等，2014：仅因摆盘不同，总体喜好度 77.2 vs. 67.6）。如果惩罚呈现方式所用的资源，岂不是在惩罚"好吃"？

DEI 框架**自然解决了这个问题**：

```
高效呈现：↑ H（更好吃），≈0 ΔE（几乎不增加资源消耗）
   → 空间排列、盘子颜色、菜名设计、环境氛围设计

低效呈现：↑ H（可能稍好吃），↑↑↑ ΔE（大幅增加资源消耗）
   → 多余餐具、被退回的"不合格"摆盘、多层装饰性包装
```

Kandinsky 研究的关键发现：**相同食材、相同盘子，仅通过重新排列——就将感知价值翻了一倍，边际资源成本为零。** 感官物理学（gastrophysics）研究表明，回报率最高的呈现策略（排列、颜色、环境设计、命名）恰恰是最便宜的。DEI 不惩罚有巧思的呈现，**它惩罚的是那些处于享乐回报曲线平坦段的高资源消耗操作**。

这个洞察本身就是一个贡献：**存在一个"呈现效率"的帕累托前沿**，而大多数餐厅都远在前沿之内。

## 4. 研究设计：三项研究

### 研究一：构建 DEI——全球食物数据库（第 1–10 个月）

**目标**：建立首个将享乐评分与环境代价配对的全球食物数据集，并正式定义 DEI。

#### 4.1.1 享乐数据收集

**路线 A — 受控感官品评（金标准）**

- **食物**：10 大菜系（中国、日本、韩国、印度、意大利、法国、墨西哥、美国、泰国、中东）中，通过分层随机抽样选取的 500 道代表性菜品
- **参与者**：2,500 名消费者（每个菜系 250 人），按 ISO 11136 要求招募
- **操作规程**：
  - 在 5 大洲 10 座城市进行中心地点测试（CLT）
  - 每位参与者对 40 道菜品评分：9 分享乐量表 + 标签享乐量表 + CATA 风味描述
  - 平衡不完全区组设计，控制认知疲劳
  - 由经过培训的厨师按标准化食谱制作
- **产出**：500 道菜品的跨文化校准享乐均值及 95% 置信区间

**路线 B — 大规模平台数据挖掘（生态验证）**

- **数据来源**：大众点评（中国）、Yelp（美国/欧洲）、Tabelog（日本）、Zomato（印度）、Google Maps（全球）的脱敏评论数据
- **目标**：50 万+ 道餐厅菜品，每道至少 30 条评论
- **NLP 管线**：
  - 微调多语言大语言模型，从评论文本中专门提取食物好吃程度的情感（与服务、环境、价格分离）
  - 将模型输出与路线 A 的受控评分进行验证（目标：菜品类别水平 r > 0.75）
- **产出**：50 万+ 道菜品的去偏享乐代理分数

#### 4.1.2 环境数据收集

- **食材级 LCA**：使用 Klimato 数据库（4,000+ 食材，20,000+ 产地变体）+ Agribalyse 3.1 + Poore & Nemecek 补充数据
- **烹饪能耗**：在 50 家合作餐厅（每个大洲 10 家）使用 IoT 能耗监测设备进行分项计量
- **清洗开销**：通过商用洗碗机的流量计测量每道菜的水和能源消耗，映射到每道菜的餐具数量
- **废弃物**：使用 AI 驱动的废弃物追踪系统（Winnow Vision 或同类产品）；辅以人工废弃物审计
- **包装**：堂食和外卖两种变体下，每份的包装重量和材料类型

#### 4.1.3 DEI 计算与验证

1. **内部一致性**：Cronbach's α 检验享乐量表信度；ICC 检验评分者间一致性
2. **构念效度**：DEI 应与现有指标（NDCI、NutriScore）相关但可区分
3. **区分效度**：DEI 应能区分领域专家判断为"高效"与"浪费"的菜品
4. **敏感性分析**：对加权参数进行蒙特卡洛模拟，评估 DEI 的稳健性
5. **分布分析**：描绘全球 DEI 分布特征——是正态？双峰？不同菜系、价格带、餐厅类型之间有何差异？

**核心假设（H1）**：*全球 DEI 分布将揭示系统性低效——相当比例的食物品项可以在显著降低环境代价的同时，保持相当的好吃程度。*

### 研究二：什么决定了 DEI？——跨国分析（第 6–14 个月）

**目标**：在菜品、餐厅、菜系和国家层面识别食物效率的结构性决定因素。

#### 4.2.1 多层次回归模型

```
DEI_ij = β₀ + β₁·菜系类型_j + β₂·烹饪方式_ij + β₃·食材数量_ij 
       + β₄·餐具数量_ij + β₅·价格水平_ij + β₆·餐厅类型_j 
       + β₇·外卖vs堂食_ij + β₈·国家监管力度_j + β₉·米其林星级_j 
       + β₁₀·绿色认证_j + u_j + ε_ij
```

- 第一层：菜品 (i)
- 第二层：餐厅 / 菜系 / 国家 (j)
- 随机截距 + 菜系类型的随机斜率

#### 4.2.2 具体假设

| 编号 | 假设 | 理论依据 | 检验方法 |
|------|------|---------|---------|
| H2a | 食材数量越多的菜品，DEI 越低 | 更多食材 → 更多供应链影响，但好吃程度不成比例增长 | β₃ < 0 |
| H2b | 每道菜的餐具数量与 DEI 呈负相关 | 额外的洗涤负荷（水 + 能源）但不改善口味 | β₄ < 0 |
| H2c | 外卖的 DEI 系统性低于堂食 | 包装废弃物增加了 E 却未改善 H | β₇ < 0 |
| H2d | 有食物浪费立法的国家（中、韩、日）平均 DEI 更高 | 监管压力减少浪费 = 降低 E | β₈ > 0 |
| H2e | 有绿色认证的餐厅 DEI 高于未认证餐厅 | 有意识的效率努力 | β₁₀ > 0 |
| H2f | 价格不是 DEI 的强预测因子 | 贵的食物不一定更高效 | β₅ ≈ 0 |
| H2g | 菜系类型解释了显著的 DEI 方差 | 烹饪传统的结构性差异 | ICC > 0.15 |
| H2h | 存在"效率前沿"——某些菜系以低代价实现高美味 | 某些食物传统天然更高效 | 帕累托分析 |

#### 4.2.3 效率前沿分析

使用数据包络分析（DEA）：
- **投入**：环境代价各维度（碳、水、能源、废弃物）
- **产出**：享乐评分
- **决策单元**：单道菜品

该分析将识别：
- **前沿菜品**：单位资源下的最大美味度（DEA 效率 = 1.0）
- **改进空间**：低效菜品在不牺牲 H 的前提下，可减少多少 E
- **菜系级效率**：哪些烹饪传统产出了最多接近前沿的菜品

---

### 研究三：DEI 信息能否改变行为？——随机对照试验（第 12–20 个月）

**目标**：通过大规模预注册 RCT 检验 DEI 标签是否影响消费者选择和餐厅运营。

#### 4.3.1 消费者实验——多地点 RCT

**设计**：2（DEI 标签：有 vs. 无）× 2（格式：数值评分 vs. A–E 分级）被试间设计，以菜系类型和价格为协变量。

**地点**：中国、美国、英国、日本 4 个国家的 20 家餐厅，每国 5 家，按菜系类型和价格带分层。

**参与者**：4,000 名就餐者（每国 1,000 人，每家餐厅 200 人），通过扫码电子菜单随机分配条件。

**实验条件**：
1. **对照组**：标准菜单
2. **DEI-数值组**：菜单每道菜旁显示 DEI 分数（如"DEI: 3.7"）
3. **DEI-等级组**：菜单每道菜旁显示等级标签（如"DEI: A ⭐"）
4. **DEI-比较组**：菜单显示相对排名（"该菜系中效率最高的 20%"）

**主要因变量**：
- 所点菜品的平均 DEI（按数量加权）
- 点选 DEI-A/B 菜品的比例
- 订单总环境代价 (E)

**次要因变量**：
- 餐后享乐满意度（9 分量表）
- 再次光顾意愿
- 感知餐厅品质
- 对高 DEI 菜品的溢价支付意愿

**核心假设（H3）**：*接触 DEI 标签将使点餐偏向更高 DEI 的菜品（实验组 vs. 对照组的平均订单 DEI 更高），同时不降低享乐满意度。*

**基准参照**：UCLA 每道菜碳标签带来了 54% 的低碳选择增加。考虑到 DEI 是一个全新指标，我们预测一个更保守的 **15–25% 的平均订单 DEI 提升**。

#### 4.3.2 餐厅运营实验

**设计**：在 10 家合作餐厅中进行前后对比，采用分阶段实施。

**干预内容**：向餐厅经营者提供：
1. 菜单上每道菜的 DEI 评分
2. 每道菜环境代价各维度的分解
3. "立竿见影的改进建议"（减少餐具数量、优化烹饪方式、减少包装层数）
4. 每月 DEI 仪表板

**结果指标**（干预前 6 个月 + 干预后 6 个月持续追踪）：
- 菜单平均 DEI
- 每位客人的总用水量
- 每位客人的总能耗
- 每位客人的废弃物（食物 + 包装）
- 每位客人的营收（确认无负面财务影响）
- 顾客满意度评分

**核心假设（H4）**：*接触 DEI 数据的餐厅经营者将实现每位客人环境代价的可测量降低（≥10%），同时顾客评分和营收不显著下降。*

**基准参照**：Champions 12.3/WRAP 数据显示，有针对性的干预在第一年可减少 26% 的浪费。

## 5. 分析方法

| 分析内容 | 工具 | 目的 |
|---------|------|------|
| 享乐数据信度 | Cronbach's α, ICC | 验证测量质量 |
| 跨文化校准 | 项目反应理论（IRT） | 在不同文化间等值享乐量表 |
| NLP 享乐情感提取 | 微调多语言 Transformer（mBERT/XLM-R） | 从评论中提取好吃程度情感 |
| DEI 计算 | 自主开发的 R/Python 包（将开源发布） | 标准化 DEI 计算 |
| 多层次回归 | lme4 (R) / statsmodels (Python) | 研究二假设检验 |
| DEA 前沿分析 | DEAP / Benchmarking (R) | 识别效率前沿 |
| RCT 分析 | 混合效应 logistic 回归、贝叶斯估计 | 研究三处理效应 |
| 蒙特卡洛敏感性 | 自定义模拟程序 | DEI 加权稳健性 |
| 结构方程模型 | lavaan (R) | 构念效度 |

## 6. 预期贡献

### 6.1 对科学的贡献

1. **首个正式框架**：将享乐测量与环境影响评估统一用于食物领域
2. **全球食物效率数据库**：首个大规模配对好吃程度与环境代价的数据库——覆盖 10 大菜系数千道菜品
3. **经验性描绘**：食物效率的分布、前沿和结构性决定因素
4. **因果证据**：效率信息是否改变食物选择行为

### 6.2 对政策的贡献

1. **气候政策的缺失变量**：当前的食物可持续性政策忽略了食物选择的首要驱动力（好吃）。DEI 提供了纳入它的工具
2. **监管应用**：DEI 分级可以像食品行业的能效标识一样运作——建立在现有框架之上（中国 GB/T 40042-2021、欧盟绿色声明指令、韩国 RFID 废弃物计费系统）
3. **平台整合**：DEI 评分可在外卖平台（美团、饿了么、DoorDash、Deliveroo）上展示，直接影响消费决策

### 6.3 对产业的贡献

1. **竞争差异化**："我们在同品类中实现最高的美味-资源比"——一种全新的营销话语
2. **运营优化**：DEI 分解揭示了具体的低效环节（如餐具数量、烹饪方式）
3. **菜单工程**：重新设计菜单以推广高 DEI 菜品，在不牺牲顾客满意度的前提下提升可持续性

### 6.4 面向大众的核心洞察

> 最高效的呈现策略——空间排列、颜色选择、菜名设计、环境氛围——恰恰是成本几乎为零的策略。最浪费的做法——多余餐具、过度包装、因外观不达标而退回的盘子——则处于享乐回报曲线的平坦段。**餐厅可以同时提升好吃程度和减少浪费。** 这不是一个权衡取舍的问题，而是一个设计问题。

## 11. 伦理考量

1. **伦理审查**：所有涉及人类参与者的研究（感官品评、消费者 RCT）需通过 IRB/伦理委员会审批
2. **不做食物羞辱**：框架明确避免将任何菜系或食物传统定义为"差的"——只识别每个传统内部的改进机会
3. **数据隐私**：所有平台评论数据均脱敏和聚合处理，不涉及个人层面追踪
4. **公平性**：承认食物效率是面向商业餐饮市场的概念——面临食物安全问题的社区不应被要求优化效率。DEI 设计用于商业餐饮，而非生存语境
5. **文化敏感性**：在每个研究地点与当地食物学者合作，确保不出现文化偏见（如暗示某种饮食传统"更高效"）
6. **开放科学**：所有数据、代码和 DEI 计算工具将在论文发表时开源发布

## 12. 论文定位

### 建议标题选项

1. **"美味效率缺口：衡量全球食物系统中单位环境代价的口味产出"**
   *"The deliciousness efficiency gap: Measuring taste output per unit of environmental cost in global food systems"*
2. **"桥接感官科学与可持续性：美味效率指数"**
   *"Bridging sensory science and sustainability: The Deliciousness Efficiency Index"*
3. **"从未被测量的指标：食物愉悦感与地球代价之比"**
   *"What nobody measured: The ratio of food pleasure to planetary cost"*

### 摘要结构建议（Nature 格式）

> 食物可持续性指标普遍将营养价值与环境影响配对，然而营养并非食物选择的首要驱动力——好吃程度才是。本文引入美味效率指数（DEI），为首个将食物享乐价值与资源消耗联系起来的标准化指标。通过对 10 大菜系 [X] 道菜品的受控感官评估以及 5 大洲 [Y] 家餐厅运营的环境影响数据，我们发现 [核心发现一：DEI 分布揭示了系统性低效]。对 [Z] 种食物品项的跨国分析显示 [核心发现二：结构性决定因素]。在 4 个国家 20 家餐厅进行的预注册随机对照试验表明 [核心发现三：DEI 标签改变了行为]。我们的结果表明，全球食物系统可以在环境代价大幅降低的同时实现相当的美味满意度——不是通过让食物变得不好吃，而是通过优化愉悦感与地球代价之比。
